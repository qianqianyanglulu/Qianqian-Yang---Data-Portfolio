# 群体分化现象的多主体建模与模拟（Schelling模型）

## 项目背景

在现实社会中，即使个体没有强烈的偏好，也可能出现明显的群体分化现象（例如居住隔离）。

本项目基于Schelling模型，模拟个体在简单偏好规则下的行为，并观察整体结构如何演化。

这个项目让我第一次接触到“复杂系统”和“涌现现象”的概念。

---

## 问题背景

假设每个个体：

- 对周围邻居有一定“相似度要求”
- 如果不满足，就会移动

问题是：**简单的局部规则，是否会导致整体上的群体分化？**

---

## 建模方法

本项目采用多主体建模（Agent-based Modeling）：

- 将每个个体视为一个 agent  
- 定义其邻居与满意度规则  
- 在网格中进行迭代更新  

每一步：

1. 检查个体是否满足条件  
2. 不满足则移动位置  
3. 重复直到系统稳定  

---

## 实现方式

- 使用 Python 构建模拟环境  
- 用二维网格表示空间分布  
- 通过循环迭代更新系统状态  
- 使用可视化展示演化过程  

---

## 🍓 结果与现象

模拟结果表明：

- 即使个体偏好较弱  
- 系统仍然会逐渐形成明显的群体分化  

这说明： **微观层面的简单规则可以导致宏观复杂结构（涌现现象）**

---

## 🍓 我的理解

这个项目让我意识到：

- 个体行为与整体结构之间存在非线性关系  
- 简单模型也可以解释复杂社会现象  
- 模拟是一种理解系统的重要方式
