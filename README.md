## 项目概览

我目前的项目主要围绕“建模与系统理解”展开，从不同角度分析现实问题：

---

### 🔹 群体分化现象的多主体建模与模拟（Schelling模型）  
（Agent-based Modeling of Segregation Dynamics）

- 构建基于规则的多主体系统  
- 模拟个体偏好如何导致群体分化  
- 分析“微观规则 → 宏观结构”的涌现现象  

关键词：复杂系统 / Agent-based simulation / 社会建模  
模拟结果表明：即使个体偏好较弱，也会产生明显的群体分化现象

---

### 🔹 潜水员平衡深度的物理建模与分析  
（Physical Modeling of Diver Equilibrium Depth）

- 基于流体力学与气体定律建立模型  
- 分析浮力与重力的动态平衡  
- 求解潜水员的自然平衡深度  

关键词：物理建模 / 连续系统 / 数值模拟  
发现存在一个关键深度，使潜水员达到浮力与重力的平衡状态

---

### 🔹 手机价格区间的统计建模与特征分析  
（Statistical Modeling of Mobile Price Classification）

- 使用逻辑回归进行分类建模  
- 对特征重要性进行分析（系数解释）  
- 探索不同变量对价格的影响  

关键词：统计学习 / 分类模型 / 特征解释  
使用逻辑回归模型进行分类，模型在测试集上达到较好的预测准确率
分析发现 RAM 等特征对价格区间具有较强影响
