#  潜水员平衡深度的物理建模与数值模拟

## 项目背景

这是我在大一期间完成的一个物理建模项目。最初只是课程中的一个题目，但我在实现过程中逐渐对“如何用数学描述现实世界”产生了很强的兴趣。因此决定记录下来。

整个项目从理解物理背景、推导公式，到用代码实现和调试，大约花了十几个小时。过程中遇到的困难主要在于：如何把抽象的物理关系转化为可以计算和可视化的模型。

---

## 题目背景

在不考虑主动运动的情况下，潜水员在水中下潜时，在理想状态下会收到两种力：

- 重力（向下）
- 浮力（向上）

由于水压会随着深度增加，肺部体积会发生变化，从而影响浮力。

因此本项目的核心问题是：**潜水员是否会在某个深度自然停住？如果会，这个深度是多少？**

---

## 🧠 建模思路

我和同学将问题拆分为三个部分来建模：

### 1️⃣ 水压模型（流体静力学）

水压随深度增加：

- 深度越深，压力越大

---

### 2️⃣ 肺部体积变化（理想气体定律）

假设肺部气体满足： - 压力 × 体积 = 常数

因此随着深度增加： - 压力增大 → 体积减小 → 浮力下降

---

### 3️⃣ 表观重量（浮力与重力）

定义：- 表观重量 = 重力 − 浮力

通过计算不同深度下的表观重量变化，判断潜水员的运动趋势。

---

## 实现过程

在实现过程中，我主要做了：

- 将物理公式转化为函数形式
- 使用 Python 进行数值计算
- 绘制不同变量随深度变化的曲线
- 分析函数变化趋势

调试过程中，我反复检查了单位、参数和函数关系，这一部分花费了较多时间。

---

## 结果与理解

通过计算和可视化，我观察到：

- 随着深度增加，肺部体积减小
- 浮力逐渐降低
- 表观重量从负变为正（或相反）

最终发现存在一个深度，使得表观重量为 0  

也就是说：潜水员会在该深度达到自然平衡（既不上浮也不下沉）
